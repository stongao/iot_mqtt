Code for Laptop 2:

The code needed to run is client_L2.py
You will need Python 2.7 or higher to execute the code. 
On linux systems, in the command window enter the following commands to install paho-mqtt library for python
1) sudo apt-get install python pip
2) pip install paho-mqtt
The following commands need to be executed to run the program.
1) python client_L2.py

You will also have to change the IP address and port in the line 68 of the code as per the broker settings.
